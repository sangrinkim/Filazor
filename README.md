# Filazor
Blazor File Explorer

IDE
- Visual Studio 2022 (Build tested: 2022-03-24)

Target framework
- ASP.NET Core 3.1
- Bootstrap v4.3.1

Icons
- Material Icons
- Open Iconic

Features
- Explore File System
- File Download / Upload
- Login (Default ID/Password: Admin/Admin1234!)
